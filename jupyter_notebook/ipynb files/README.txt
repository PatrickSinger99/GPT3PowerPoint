Running the script
------------------
- Create a txt file in this directory, name it "openai_key.txt" and copy your openai key into it
- run the file "MAIN_gpt_markdown_creator.ipynb"

Installing modules
------------------
- openai = pip install openai
- gpt (is already in project as "gpt.py") = https://github.com/shreyashankar/gpt3-sandbox/blob/master/api/gpt.py
