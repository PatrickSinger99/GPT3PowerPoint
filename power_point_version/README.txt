Running the scripts
------------------
- Create a txt file in the version directory, name it "openai_key.txt" and copy your openai key into it

Version 1:
- Run gpt_power_point_data.ipynb
Version 2:
- Run gpt_power_point_data_mul_slides.ipynb
Version 3:
- Run gpt_summarization_without_input.ipynb

Installing modules
------------------
- openai = pip install openai
- Python Module = pip install python-pptx
- Wikipedia API = pip install wikipedia-api
- Google Image Downloader = pip install git+https://github.com/Joeclinton1/google-images-download.git
- gpt (is already in project as "gpt.py") = https://github.com/shreyashankar/gpt3-sandbox/blob/master/api/gpt.py
